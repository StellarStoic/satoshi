Instructions for Uploading Your Translated Bitcoin Whitepaper

To upload your translated whitepaper, please follow these steps:

  • Ensure your translation is accurate and complete, and save it in PDF format.

  • Fork this repository on GitHub: https://github.com/StellarStoic/satoshi

  • Scan the PDF file with VirusTotal to ensure it is safe. Once confirmed, copy the relevant data into the `whitepaper/whitepapers.json` file.

  • Update the whitepapers.json file:
    • Open the whitepaper folder.
    • Add or update entries as necessary.

  • Name your PDF file in the format: bitcoin_{lower cased ISO_alpha-2_country_code}.pdf (e.g., bitcoin_sl.pdf).

  • Include your PDF file in the appropriate folder named after the ISO alpha-2 country code representing the language's country (e.g., LT for Lithuania).

  • Find ISO alpha-2 country codes online if you're unsure about the correct code.

  • Once you've made your changes, submit a pull request to merge your updates into the main repository.

Thank you for contributing to the Bitcoin community!
